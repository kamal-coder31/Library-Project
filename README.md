# Library Management System

## Project Description
This project is a Library Management System developed using Python.
It allows users to manage books, borrowers, and library operations.
This system is designed using object-oriented programming concepts such as classes and modules.


## Features
- Add new books
- View available books
- Register users
- Borrow books
- Return books
- Menu-driven interface

## How to run the program
1. Open the project folder in VS code or terminal.
2. Run the main file:
          python main.py
3. Follow the menu options displayed.


## Project Structure
- book.py - Handles book details
- user,py - Handles user details
- borrower.py - Borrowing functionality
- library.py - Core library logic
- main.py - Runs the program

## Example Usage
1. Select option 1 to add a book.
2. Select option 2 to view a book.
3. Select option 3 to borrow a book.
